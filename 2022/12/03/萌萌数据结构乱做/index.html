【笔记】萌萌数据结构乱做

发表于 2022-12-03 更新于 2022-12-28 分类于 笔记

本文记录了菜狗 wcy 在学习 whk 期间用于放松身心的极简单 DS 题(暂时 AFO，滚回 whk 了)。

Luogu3797

题目传送门

显然线段树，考虑在每个节点维护以下信息:

  • \(sum\)，该区间答案。
  • \(lp_{0/1}\)，该区间最左端 ( 或 ) 的位置，没有记为 \(n + 1\)。
  • \(rp_{0/1}\)，该区间最右端 ( 或 ) 的位置，没有记为 \(0\)。

显然我们有 \(u.sum = l.sum + r.sum + [l.rp_1 > l.rp_0][r.lp_0 < r.lp_1]\)，可以进行 \(O(1)\) 转移，查询时直接合并区间信息即可。时间复杂度 \(O(n\log n)\)。

Luogu3203 & CF13E

题目传送门

题目传送门

首先我们对原序列进行分块，块长为 \(\sqrt{n}\)，然后维护两个东西 \(f_i\) 和 \(g_i\)，分别表示几次跳出块和跳出块后在哪里，从后往前扫一遍可以 \(O(n)\) 求。

对于每个查询操作，我们可以每次暴力跳块 \(O(\sqrt{n})\) 求出答案。

对于每次修改操作，我们直接每次对当前快进行暴力重构，是 \(O(\sqrt{n})\) 的。

CF13E 需要再维护一个 \(h_i\)，表示跳出块最后一次在哪里。

时间复杂度 \(O(n) - O(\sqrt{n})\)。

CF739C

题目传送门

思维简单，考验 debug 能力的线段树。

考虑维护以下信息:

  • \(lp, rp\)，该区间左右端点。
  • \(lmax, rmax\)，以该区间左端点为起始的最长下降段，以该区间右端点为结束的最长上升段。
  • \(kmax_{0/1}\)，以该区间左右端点为起始或结束的最长先升再降子段。
  • \(tmax\)，该区间答案。
  • \(tag\) 该区间加法懒标记。

转移和下传懒标记都很平凡，查询直接输出全局 \(tmax\) 即可，时间复杂度 \(O(n\log n)\)。

1204校内模拟赛 T3

因为给的是一棵树，所以可以得到 连通块 = 点数 - 边数，所以直接求保留边数即可。

考虑用 \((x, y, z)\) 定义一条边，则这条边保留等价于 \(x, y, z \in [l, r]\)，但这样变成了三维数点，不好写且复杂度高，所以考虑令 \(a = \max(x, y, z), b = \min(x, y, z)\)，这样就变为了 \(a, b \in [l, r]\)，直接二维数点即可，时间复杂度 \(O(n\log n)\)。

CF817E

题目传送门

01-Trie 套路题，前两个操作直接在 Trie 上修改即可，重点在于如何处理查询。

首先令 \(p, l\) 的第 \(i\) 为分别为 \(p_i, l_i\)，那么我们发现当 \(l_i = 1\) 时，我们当前位异或上 \(p_i\) 即可满足要求，所以可以加上一整个子树的贡献，然后去到另一子树，而当 \(l_i = 0\) 时，我们就什么都算不了，只能去到 \(p_i\) 这一子树，在 Trie 树上统计答案，时间复杂度 \(O(n\log p)\)。

CF1311F

题目传送门

注意到若 \(x_i < x_j, v_i >v_j\)，则 \(d(i, j) = 0\)，所以问题转化为求 \[ \sum_{i, j, i \neq j}^n [x_i \le x_j][v_i \le v_j](x_j - x_i) \] 直接开两个 BIT 搞二维偏序即可，时间复杂度 \(O(n\log n)\)。

Luogu4247

题目传送门

一直想做的一道 DS，在数学课上推出来了。

观察到 \(c\) 很小，所以考虑对于每一个 \(c\) 都在线段树上记录它的答案，所以我们维护以下信息

  • \(f_i\)，当前区间 \(c = i\) 时的答案。
  • \(tag, rev\)，两个标记。

显然 pushup 是好做的，因为有 \[ u.f_i = \sum_{j + k = i} l.f_j \cdot r.f_k \] 然后区间取反也是好写的，只需暴力修改当 \(c\) 为奇数时的答案即可。

所以考虑如何处理区间加，我们不妨从小数据开始考虑，观察到 \[ (a_1 + k)(a_2 +k) + (a_2 + k)(a_3 + k) + (a_1 + k)(a_3 + k) = a_1a_2a_3 + 2k(a_1 + a_2 + a_3) + k^2 \] 这其实我们用幂和组合数去搞，然后我们稍微推推就可以得到 \[ u.f_i = \sum_j^{i - 1} c^{i - j}\binom{u.r - u.l +1 - j}{i - j}u.f_j \] 注意到小的 \(f\) 对大的 \(f\) 有影响，所以从大到小更新，注意实现需要精细，应预处理 \(c\) 的幂而不是快速幂。

时间复杂度 \(O(c^2m\log n)\)。

Luogu3431

题目传送门

第一道在学校做的 OI 题。

观察到 \(k\) 很小，所以进行离散化，注意多加一个车站 \((n, m)\)，然后以 \(x\) 为第一关键字，\(y\) 为第二关键字排序，然后令 \(f_i\) 为走到第 \(i\) 个车站的最大值，不难得到转移 \[ f_i = \max_{j\in[1, i - 1], y_j \le y_i}\{f_j\} + a_i \] 直接做是 \(O(n^2)\) 的，所以用线段树类似二维偏序的方式维护前面的东西，可以做到 \(O(n\log n)\)。

CF833B

题目传送门

考虑 \(f_{i, j}\) 为将 \([1, j]\) 划分为 \(i\) 段的最大值，显然有转移 \[ f_{i, j} = \max_{k \in [1, j]} f_{i - 1, k - 1} + cnt_{k, j} \] 其中 \(cnt_{i, j}\) 表示区间 \([i, j]\) 的价值。

不难想到线段树优化，在处理一个新的 \(i\) 值时先让线段树继承上一层的 \(f\)，然后令 \(pre_i\) 表示 \(a_i\) 上一次出现的位置，这样每次把 \([pre_j + 1, j]\) 中的答案加一，就变成了一个区间加区间最大值问题，可以 \(O(nk\log n)\) 解决。

CF522D & 1211校内模拟赛 T2

题目传送门

考虑离线，将询问按右端点进行分组。

预处理 \(pre_i\) 表示 \(a_i\) 上一次出现的位置，然后我们考虑用一棵线段树来维护每个位置上的答案。

考虑插进一个数 \(i\) 会对答案造成什么影响，显然我们要将 \(pre_i\) 的答案改为 \(i - pre_i\)，然后处理以 \(i\) 为右端点的询问，直接区间最小值即可，时间复杂度 \(O(m\log n)\)。

Luogu1975

题目传送门

虽然能暴力，但还是当成树套树板子做的。

首先离散化然后搞个 BIT 求出全局逆序对，然后考虑交换 \(a_l, a_r (l < r)\) 会对全局答案造成那些影响。

形式化地，记 \(f_{l, r, x} = \sum_{i = l} ^ r [a_i < x], g_{l, r, x} = \sum_{i = l} ^ r [a_i > x]\)，我们首先统计 \((l, r)\) 对全局答案的贡献，手玩一下发现是 \[ g_{l + 1, r - 1, a_l} - f_{l + 1, r - 1, a_l} + f_{l + 1, r - 1, a_r} - g_{l + 1, r - 1, a_r} \] 对于 \(a_l, a_r\) 这两个数产生的贡献直接比较大小即可，由于带修，所以用线段树套平衡树做到 \(O(n\log^2n)\)。

Luogu4093

题目传送门

考虑将数组中的每一个元素的数用一个三元组 \((a, l, r)\)，表示该位置的值，可变化的值的最小值，可变化的值的最大值，然后设计状态 \(f_i\) 为前 \(i\) 个数中选第 \(i\) 个数的最长序列长度，显然有转移 \[ f_i = \max_{j \in [1, i) \land r_j \le a_i \land a_j \le l_i} f_j + 1 \] 直接做是 \(O(n^2)\) 的，可以用 CDQ 分治优化到 \(O(n\log^2 n)\)，注意在使用 CDQ 分治优化 DP 时，一定要先做完左半区间然后按 \(id\) 排序做右半区间。

Luogu3364

题目传送门

首先对原序列按照 \(l\) 进行排序。

设计状态 \(f_i\) 表示前 \(i\) 个人中选第 \(i\) 个的答案，那么依题意有 \[ f_i = \max_{j \in [1, i) \land a_j \le s_i \land w_j \le a_i}f_j + 1 \] 用 CDQ 优化至 \(O(n\log^2 n)\)。

Luogu4390

题目传送门

记 \(\{(a, b), (c, d)\}\) 为以 \((a, b)\) 为左下角，\((c, d)\) 为右上角的子矩阵的和。

首先利用二维前缀和思想把 \(\{(x_1, y_1), (x_2, y_2)\}\) 拆成 \[ \{(1, 1), (x_2, y_2)\} + \{(1, 1), (x_1 -1, y_1 - 1)\} - \{(1, 1), (x_1 - 1, y_2)\} - \{(1, 1), (x_2, y_1 - 1)\} \]

那么就变成了求以 \((x, y)\) 为右上角的矩阵的和，是个简单的二维数点，接下来处理带修，给每一个修改或询问加上一个时间戳，这样是一个三维偏序，用 CDQ 解决即可，时间复杂度 \(O(n\log^2 n)\)。

Luogu2487

题目传送门

思路一眼，但细节很多，写起来难受的一道题。

记 \(f_i\) 为以 \(i\) 结尾的最长子序列，\(g_i\) 为这样子序列的个数，那么显然有转移 \[ f_i = \max_{j \in [1, i)\land h_j \ge h_i \land v_j \ge v_i} f_j + 1\\ g_i = \sum_{j = 1}^{i - 1} g_j[f_j + 1 = f_i] \] 因为我们要计算经过 \(i\) 的子序列个数，所以要再从后往前 DP 一遍，这样用 CDQ 分治优化至 \(O(n\log^2 n)\)。

1225作业T2

题目所求式子为 \[ \sum_{l=1}^n\sum_{r=l}^n\sum_{i=l}^r\sum_{j=i+1}^r a_ia_j[a_i > a_j] \] 如果我们直接枚举 \([l, r]\)，那么时间复杂度至少来到 \(O(n^2)\)，必然不行，所以我们倒着考虑每一对逆序对 \((i, j)\) 造成的贡献为多少，也就是说有多少个区间包含 \([i, j]\)，这个的答案显然为 \(i(n - j + 1)\)，所以每一对逆序对 \((i, j)\) 造成的贡献为\(i(n - j +1)a_ia_j\)，然后把式子重写一下就变成 \[ \sum_{i=1}^n\sum_{j=i+1}^n i(n - j + 1)a_ia_j[a_i > a_j] = \sum_{i=1}^n ia_i\sum_{j=i+1}^n(n - j +1)a_j[a_i > a_j] \] 直接 BIT 维护逆序对即可，时间复杂度 \(O(n\log n)\) (未包括龟速乘复杂度)。

CF609E

题目传送门

一个 MST 的经典 trick，我们首先可以建出全局 MST，然后我们考虑加进一条边 \((u, v)\)，这样我们会发现这棵树变成了基环树，所以我们需要断掉环上的一条边，由于我们需要权值最小，所以肯定要断掉 MST 上 \(u \rightarrow v\) 边的最大值，树链剖分维护即可，时间复杂度 \(O(m\log n)\)。(注意到由于作者太菜没写 ST 表，所以时间复杂度变为 \(O(m\log^2 n)\) )。

CF1567E

题目传送门

考虑线段树，维护以下信息

  • \(lp, rp\)，区间左右端点。
  • \(lmax, rmax\)，区间左/右侧 LIS 长度。
  • \(sum\)，区间答案。

转移平凡，时间复杂度 \(O(n\log n)\)。

ABC283F

题目传送门

纪念第一场打到 G 的 ABC（这 F 不比 E 简单多了。

看到绝对值显然是个分讨，因此我们有 \[ D_i = \min_{i\neq j}(|P_i - P_j| + |i - j|) = \begin{cases} \min(P_i + i - P_j - j) & P_i > P_j \land i > j\\ \min(i - P_i - j + P_j) & P_i < P_j \land i > j\\ \min(P_i - i + j - P_j) & P_i > P_j \land i < j\\ \min(-P_i - i + j + P_j) & P_i < P_j \land i < j \end{cases} \] 对着四种情况分别做一遍二维偏序即可，时间复杂度 \(O(n\log n)\)。

Luogu8575

题目传送门

子树问题显然用 dfs 序拍到序列上做，我们用 \((Red_i, Blue_i, l_i, r_i)\) 来描述一个点，分别表示红星、蓝星、dfs 序起始点，那么点 \(j\) 对点 \(i\) 有贡献当且仅当 \(Red_j \le Red_i \land Blue_j \le Blue_i \land l_j \in [l_i, r_i]\)，直接 CDQ 治即可，时间复杂度 \(O(n\log^2 n)\)。
