【笔记】成套方法

发表于 2022-06-15 更新于 2022-06-25 分类于 笔记

前言

最近开始看《具体数学》，看到第一章的成套方法，感觉打开新世界大门，就来记录一下这一种美妙的方法。

成套方法的概述

成套方法主要是用来解递归式的，这种方法技巧性较强，思路非常清奇，具体步骤可以用两个步骤概括

  1. 化特殊为一般
  2. 化一般为特殊

具体我们来看下面一道例题

解递归式 \[ \begin{cases} t_0 = 1 \\ t_n = t_{n - 1} + 2n + 3 (n > 0)\\ \end{cases} \] 首先我们运用化特殊为一般的思想，将原递归式转化成这个样子 \[ \begin{cases} t_0 = x \\ t_n = t_{n - 1} + yn +z (n > 0) \\ \end{cases} \] 显然，我我们的 \(t_n\) 肯定只由 \(x,y,z\) 组成，所以我们不妨设 \[ t_n = X(n)x + Y(n)y + Z(n)z \] 观察到，我们 \(x, y, z\) 的取值并不影响 \(X(n),Y(n),Z(n)\) 的取值，所以我们可以使用类似特殊值法的方法来求解 \(X(n),Y(n),Z(n)\)，不过成套方法给出了一种更巧妙的方法，那就是假设 \(t_n\) 的值，反推出 \(x,y,z\) 的值

我们先令 \(t_n = 1\)

那我们可以得到 \[ \begin{cases} 1 = x \\ 1 = 1 + yn + z \\ \end{cases} \] 那我们就得 \[ \begin{cases} x = 1 \\ y = 0 \\ z = 0 \\ \end{cases} \] 所以我们就有 \(X(n) = 1\)

再令 \(t_n = n\)，有 \[ \begin{cases} 0 = x \\ n = n - 1 + yn +z \end{cases} \] 得到 \[ \begin{cases} x = 0 \\ y = 0 \\ z = 1 \end{cases} \] 轻松得到 \(Z(n) = n\)

最后，我们令 \(t_n = n^2\) ，得到 \[ \begin{cases} 0 = x \\ n^2 = (n - 1)^2 + yn + z \end{cases} \] 得到 \[ \begin{cases} x = 0 \\ y = 2 \\ z = -1 \end{cases} \] 带回去，得到 \[ n^2 = 2Y(n) - Z(n) \\ Y(n) = \frac{n(n+ 1)}{2} \] 所以，我们可以得到 \[ t_n = x + \frac{n(n +1)}{2}y +nz \] 将 \(x = 1,y = 2,z = 3\) 代入原式中，得到 \[ t_n = 1 + n(n +1) +3n = n^2 +4n +1 \]

取求值点的技巧

在成套方法中，我们最具有技巧性的一步就是巧妙地取 \(t_n\) 的值，这里以一个例子来讲述一下如何取到合适的值。

首先我们要记住取值的目的：消元，来看下面这个例子。 \[ \begin{cases} t_1 = 1 \\ t_{2n} = 2t_n - 1 \\ t_{2n + 1} = 2t_n +1 \\ \end{cases} \] 这个递归式其实就是约瑟夫问题的递归式，我们还是采用成套方法的思想，先转化递归式。 \[ \begin{cases} t_1 = x \\ t_{2n} = 2t_n +y \\ t_{2n + 1} = 2t_n + z \\ \end{cases} \] 然后设 \[ t_n = X(n)x +Y(n)y + Z(n)z \] 首先令 \(t_n = 1\)，我们有 \[ \begin{cases} x = 1 \\ y = -1 \\ z = -1 \\ \end{cases} \] 那么也就有 \[ X(n) - Y(n) - Z(n) = 1 \] 再令 \(t_n = n\)，得到 \[ \begin{cases} x = 1 \\ y = 0 \\ z = 1 \\ \end{cases} \] 代回去得到 \[ X(n) + Z(n) = n \] 但是我们现在好像陷入了瓶颈，无法求出一个具体的系数，为了解决这个问题，我们考虑取一个 \(t_n\) 使得 \(y = z = 0\)

这样我们可以得出 \(x\)，也就可以求出 \(X(n)\)

如果我们想要 \(y = z = 0\)，那么我们就有 \[ \begin{cases} t_{2n} = 2t_n \\ t_{2n + 1} = 2t_n \\ \end{cases} \] 观察到函数 \(t_n = 2^m\) 满足要求，其中 \(2^m \le n < 2^{m + 1}\)，也即 \(m = \lfloor \log_2^n \rfloor\)

所以我们取 \(t_n = 2^m\)，得 \[ \begin{cases} x = 1 \\ y = 0 \\ z = 0 \\ \end{cases} \] 轻松得到 \(X(n) = 2^m\)

然后根据我们前文得到的方程组，我们可以求出 \[ \begin{cases} X(n) = 2^m \\ Y(n) = 2^{m +1} - n - 1\\ Z(n) = n - 2^m \end{cases} \] 最后将 \[ \begin{cases} x = 1 \\ y = -1 \\ z = 1 \\ \end{cases} \] 代入，我们可以得到 \[ t_n = 2n - 2^{m + 1} +1 \] 我们也就解决了这个问题

总结

成套方法是一种巧妙解决递归式的方法，但是我们遇到复杂的递归式，还是更加常用生成函数求解。
