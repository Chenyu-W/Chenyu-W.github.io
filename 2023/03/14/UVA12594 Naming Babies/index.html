【题解】UVA12594 Naming Babies

发表于 2023-03-14 更新于 2023-03-18 分类于 题解

题目传送门

太久没写 DP 了，于是找了一道简单的斜率优化练手题来做。

分析

观察每个字符的花费 \((i - pos_i) \cdot pos_i = i \cdot pos_i - pos_i \cdot pos_i\)，后面的 \(pos_i \cdot pos_i\) 无论如何划分都不会改变，所以我们只需要求划分后 \(\sum_{i} i\cdot pos_i\) 的最小值即可。

考虑对每一段划分的花费进行拆分，即如果我们把 \([l, r]\) 化为一组，那么这一组的花费为 \(\sum_{i = l}^r (i - l)pos_i\)，我们记 \(a_i = \sum_{j = 1}^i j\cdot pos_j\)，\(b_i = \sum_{j = 1}^i pos_j\)。这样我们就有 \(\sum_{i = l}^r(i - l)pos_i = a_r - a_{l - 1} - l\cdot b_r + l\cdot b_{l - 1}\)，这样方便我们 DP。

设 \(f_{j, i}\) 为前 \(i\) 个数划分为 \(j\) 段的最小值，根据上式，我们容易得到 \[ f_{j, i} = \min_k f_{j - 1, k} + a_i - a_k - (k + 1)b_i + (k + 1)b_k \] 这是一个斜率优化的式子，因为这是斜率优化入门题，所以讲一下具体如何优化。

设 \(k_1 < k_2\) 且 \(k_2\) 优于 \(k_1\)，那么我们就有 \[ f_{j - 1, k_2} + (k_2 + 1)b_{k_2} - a_{k_2} - (k_2 + 1)b_i \le f_{j - 1, k_1} + (k_1 + 1)b_{k_1} - a_{k_1} - (k_1 + 1)b_i \] 设 \(Y(j, i) = f_{j - 1, i} + (i + 1)b_i - a_i\)，\(X(i) = i\)。

化简得到 \[ b_i \ge \dfrac{Y(j, k_2) - Y(j, k_1)}{X(k_2) - X(k_1)} \] 因为 \(X\) 和 \(b_i\) 都是单调递增的，所以可以直接单调队列维护凸包，时间复杂度 \(O(nk)\)。
