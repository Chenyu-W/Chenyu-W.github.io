【笔记】基础函数小记

浅谈高一数学中函数的学习(不包括三角函数)，笔者很菜，轻喷。

函数的定义和表示

初中的时候我们学习了基本的一次函数和二次函数，下面我们给出集合论中函数的定义，函数的本质其实是一种映射关系，如果我们有一个集合 \(A\)，一个集合 \(B\)，对于 \(\forall x \in A\)，都有唯一一个 \(y \in B\) 使得 \(f(x) = y\)，那么称 \(f: A \rightarrow B\) 是一个函数。

函数的定义域其实就是函数中自变量的取值范围，比如 \(f(x) = \dfrac{2}{x - 1}\) 的定义域为 \((-\infty, 1) \cup (1, +\infty)\)，这是好理解的，而函数的值域就是因变量的取值范围，比如 \(f(x) = x^2 - 2x + 1\) 的值域为 \([0, +\infty)\)，这都是好理解的。

判断两个函数是否是同一函数需要两步，一是判断定义域是否相同，二是判断映射关系是否相同。比如都定义在 \(\{1, 2\}\) 上的两个函数 \(f(x) = x^2\) 和 \(g(x) = 3x - 2\) 是一个函数因为 \(f(x)\) 和 \(g(x)\) 的定义域相同，而且 \(f(x)\) 和 \(g(x)\) 都让 \(1 \rightarrow 1\)，\(2 \rightarrow 4\)，所以判断相等。

更加常见的题是求函数的定义域和值域，求定义域只要根据函数中的特殊部分如分式、根式进行计算。而求值域比较复杂，可能需要应用单调性和不等式的知识。

函数的单调性

初中时我们知道 \(y = kx + b\) 在 \(k > 0\) 时单调递增，在 \(k < 0\) 时单调递减，对于函数的单调性，在高中阶段有更严谨的定义。

对于一个函数 \(f(x)\) 来说，若对其定义域内的某个区间 \(D\)，\(\forall x_1 < x_2 \in D\)，都有 \(f(x_1) < f(x_2)\)，那我们称函数 \(f(x)\) 在 \(D\) 上是增函数，减函数的定义与其类似，只需将 \(f(x_1) < f(x_2)\) 改为 \(f(x_1) > f(x_2)\) 即可。特别地，如果 \(f(x)\) 在其定义域上有单调性，则称 \(f(x)\) 为增函数/减函数。

用定义证明函数单调性一般有以下几个步骤，首先 \(\forall x_1 < x_2 \in D\)，然后判断 \(f(x_2) - f(x_1)\) 的符号，如果是正号就是增函数，否则是减函数。当然遇到一些简单的函数时，我们并不需要利用定义证明，只需简单判断即可。

接下来为了方便说明，我们用 \(\uparrow\) 代表增函数， \(\downarrow\) 代表减函数。

首先如果我们要判断两个函数进行加减运算的单调性，显然地，我们有 \(\uparrow + \uparrow = \uparrow\)，\(\downarrow + \downarrow = \downarrow\)，\(-\downarrow = \uparrow\)，\(-\uparrow = \downarrow\)。

接下来我们来讨论复合函数的单调性，首先对于只有两个函数的简单复合函数，我们显然有 \(\uparrow(\uparrow) = \uparrow\)，\(\downarrow(\downarrow) = \uparrow\)，\(\uparrow(\downarrow) = \downarrow\)，\(\downarrow(\uparrow) = \downarrow\)，然后我们可以推广到 \(n\) 个函数。具体地，设函数 \(f_1(x), f_2(x), \ldots, f_n(x)\) 都在定义域上单调，那么对于函数 \(f_1(f_2(\cdots f_n(x))\)，其单调性取决于这 \(n\) 个函数中减函数的奇偶，如果有奇数个，则这个函数为减函数，否则为增函数，对 \(n\) 应用归纳容易证明。复合函数的单调性有时能带给我们很大的帮助，比如已知 \(f(x)\) 是减函数，要求 \(f(g(x))\) 的单调递增区间，那么问题就等价于求 \(g(x)\) 的单调递减区间。
