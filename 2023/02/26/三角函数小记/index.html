【笔记】三角函数小记

浅谈一些高中三角函数的基础内容(笔者很菜，轻喷)。

角的概念与弧度制

初中的时候，我们学过的角是一个静态的概念，即同端点的两条射线构成的图形，到了高中阶段，这个概念变得更加动态，我们定义角是一条射线绕其断点旋转形成的图形，如果是逆时针旋转，那么就成为正角，顺时针则称为负角，有了这个定义，我们就会出现一些初中阶段不可能出现的角，比如 $\angle A = -30^\circ$，这样角的定义就推广到了任意角，我们定义旋转之前的那条射线称为角的始边，而旋转之后形成的那条射线称为角的终边。

细心地你肯定发现了，不同度数的角可能终边相同，比如 $330^\circ, -30^\circ$ 的终边就是一样的。我们稍作思考，发现终边相同相当于这条射线多旋转了若干个 $360^\circ$，所以我们容易得到与 $\alpha$ 终边相同的角的集合为 $S = {\alpha + k \cdot360^\circ | k \in \mathbf Z}$、

接下来我们引入象限角的概念，假设我们有一个平面直角坐标系，然后我们规定角的始边 $OA$ 在 $x$ 轴非负半轴上，然后我们规定角的终边 $OB$ 在哪个象限，就称 $\angle AOB$ 为第几象限角，比如 $\alpha = 45^\circ$ 为第一象限角。

为了方便表示，引入弧度制的概念，我们规定 $\pi\text{rad} = 180^\circ$，其中 $\text{rad}$ 通常可省略不写，一些常见的转换：$\dfrac{\pi}{2} = 90^\circ$, $\dfrac{\pi}{6} = 30^\circ$ 等。我们发现弧度制在扇形中有一些很好的结论，假设扇形的弧长为 $l$，半径为 $r$，那我们可以得到圆心角 $\alpha = \dfrac{l}{r}$，然后我们可以得到扇形面积 $S = \dfrac{1}{2}\alpha r^2 = \dfrac{1}{2}lr$ 这些都是很好的结论。

三角函数与同角三角比

定义单位圆为圆心在原点，半径 $r = 1$ 的圆。设点 $A(1, 0)$，将 $OA$ 绕点 $O$ 旋转 $\alpha$，得到的射线与单位圆交于 $B(x, y)$，则我们有 $\sin \alpha = y, \cos \alpha = x, \tan\alpha = \dfrac{y}{x}$，特别地，当 $x = 0 $ 时，即 $\alpha = \dfrac{2k + 1}{2}\pi(k \in \mathbf Z)$，$\tan \alpha$ 不存在，不难发现这个定义和初中阶段学的大体相似。

一些常见角度的三角函数

根据勾股定理的知识，我们可以得到 $\sin^2 \alpha + \cos^2 \alpha = 1$，同时根据定义可以得到 $\tan \alpha = \dfrac{\sin \alpha}{\cos \alpha}$，根据这些关系，我们不难做到知一求二，一个常见的技巧是 $\sin \alpha \pm \cos \alpha = \pm k \Leftrightarrow 1 \pm 2 \sin\alpha\cos\alpha = k^2$。

诱导公式

奇变偶不变，符号看象限。

上面这句话很好地概括了诱导公式的主要内容，下面我们来介绍诱导公式的使用。

假设我们要化简 $\sin\left(\alpha \pm\dfrac{k}{2}\pi\right)$ 或 $\cos\left(\alpha \pm\dfrac{k}{2}\pi\right)$，可以使用诱导公式，比如 $\sin\left(\dfrac{\pi}{2} - \alpha\right) = \cos(\alpha)$，$\cos\left(\pi - \alpha\right) = -\cos\alpha$。首先先解释前半句话，这里的奇偶指的是 $k$ 的奇偶，即如果 $k$ 为奇数，三角函数名称发生变化，$\sin \rightarrow \cos, \cos \rightarrow \sin$，反之则不变。而后半句话我们可以这样实操，我们假设 $\alpha \in \left(0, \dfrac{\pi}{2}\right)$，我们在单位圆上表示出 $\alpha \pm \dfrac{k}{2}\pi$，然后看它的象限，再用原函数在此象限的符号给结果定号。For example，我们要计算 $\sin\left(\alpha - \dfrac{\pi}{2}\right)$，首先这里 $k$ 是奇数，所以最后的结果肯定是 $\cos$，然后假设 $\alpha \in \left(0, \dfrac{\pi}{2}\right)$，那么 $\alpha - \dfrac{\pi}{2}$ 为第四象限角，$\sin$ 在此象限为负，所以最终的结果为 $-\cos \alpha$。

三角恒等变换

多公式预警！！！

上述是最基础的两角和差公式，经常在一些推式子题中出现，由两角和的特殊形式还可以推导出倍角公式。

在做题的过程中，这些公式一般有两种用法，一种是正用，也就是知道前面的，把它展开成后面那样；另一种则是逆用，通过诱导公式把式子化成上述公式的形式，使得式子易于计算，需要做题训练。

倍角公式有一些重要的推论，在实际题目中有很大的作用。

其中第二条和第三条可以在一些需要降幂或者带根号的式子中使用。

此外，倍角公式还有一个额外的 trick，就是求解 $\cos\alpha\cos2\alpha\cos4\alpha\cdots$，以求解 $\cos\alpha\cos2\alpha\cos4\alpha$ 为例，我们先上下同乘 $\sin\alpha$ 得到 $\dfrac{\sin\alpha\cos\alpha\cos2\alpha\cos4\alpha}{\sin\alpha}$，然后我们惊奇的发现了一个和正弦二倍角公式长得差不多的东西，于是我们把式子变成 $\dfrac{\frac{1}{2}\sin2\alpha\cos2\alpha\cos4\alpha}{\sin\alpha}$，然后这个式子又可以同上进行转化，最终我们可以化简为 $\dfrac{1}{8}\dfrac{\sin8\alpha}{\sin\alpha}$，这样就会简单一些。

二倍角有一个重要的推论，即半角公式。

倍角公式除了二倍角还有三倍角，虽然不常见，但还是看一下。

这些式子从左往右推难度不大，在实际应用中更多的是从右边的形式化成三倍角形式。

最后是万能公式，就是用含有 $\tan\dfrac{\alpha}{2}$ 的式子表示三角函数。

辅助角公式主要求解如下问题：将 $a\sin\alpha + b\cos\alpha$ 化为 $A\sin(\omega\alpha + \varphi)$ 的形式。

首先我们考虑将这个化成 $\sin$ 和公式的形式，给 $\sin\alpha$ 和 $\cos\alpha$ 配两个系数 $x, y$，满足 $x^2 + y^2 = 1$，这样我们就可以把这个化成 $\sin$ 和的形式，观察到 $a\sin\alpha + b\sin\alpha = \sqrt{a^2+b^2}(\dfrac{a}{\sqrt{a^2+b^2}}\sin\alpha + \dfrac{b}{\sqrt{a^2+b^2}}\cos\alpha)$，然后我们发现配的系数就满足我们的要求，所以我们肯定能找到一个角 $\varphi$，使得 $\cos\varphi = \dfrac{a}{\sqrt{a^2+b^2}}$，$\sin\varphi = \dfrac{b}{\sqrt{a^2+b^2}}$，然后原式就变为了 $\sqrt{a^2+b^2}\sin(\alpha + \varphi)$。

观察上表，我们发现这两个函数有非常多相似之处，比如我们发现 $f(x) = \cos x$ 的对称轴和对称中心都可以由 $f(x) = \sin x$ 的向右平移 $\dfrac{\pi}{2}$ 得出。这本质上是因为 $\sin\left(\dfrac{\pi}{2} + x\right) = \cos x$，所以其实这两个函数图像可以互相通过平移得到。

了解完了最简单的三角函数，接下来我们来看一类正弦型三角函数的图像，即 $f(x) = A\sin(\omega x + \varphi)$。

对于函数 $f(x) = A\sin(\omega x + \varphi)$，$A$ 被称为振幅，$\omega$ 被称为频率，$\varphi$ 被称为初相。学过函数图像变换的我们很容易得到这个图像相当于是 $f(x) = \sin x$ 的图像纵向伸缩 $A$ 倍，横向伸缩 $\dfrac{1}{\omega}$ 倍，然后向左/右平移 $\left|\dfrac{\varphi}{\omega}\right|$ 得到的，这样我们就很容易画出这个的函数的图像了，同时如果 $f(x)$ 的最小正周期为 $T$，我们不难得到 $\omega = \dfrac{2\pi}{T}$
