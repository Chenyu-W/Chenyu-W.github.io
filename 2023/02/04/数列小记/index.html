【笔记】数列小记

浅谈一下数列的基础内容(笔者水平不高，轻喷)。

等差数列

定义

如果数列 \(\{a_n\}\) 满足 \(a_{n + 1} - a_n\) 是常数，那么则称数列 \(\{a_n\}\) 是一个等差数列，等差数列还可以写作 \(\text{A.P}\)。

等差数列的基本量

\(a_1\) 称为首项，\(a_{n + 1} - a_n\) 这个常数称为公差，一般用 \(d\) 表示。

有了以上定义，我们可以容易得到等差数列的通项公式 \(a_n = a_1 + (n - 1)d\)。

我们还可以发现把一个等差数列的头尾两两配对，它们的和是一样的，这也是高斯计算等差数列和的原理，从中我们可以推出等差数列的前 \(n\) 项和公式 \(S_n = \dfrac{n(a_1 + a_n)}{2}\)，有时也用通项公式换掉 \(a_n\) 来方便计算。

等差数列的性质

等差数列在 \(d \neq 0\) 时是单调的，看起来十分废话，但是这条结论在一类求 \(S_n\) 的最值问题可以应用，我们只需找到 \(a_n\) 和 \(0\) 的分界点即可。

等差数列的通项公式有一个弊端，那就是只能从 \(a_1\) 推 \(a_n\)，好在我们可以把通项公式变形一下，这样就可以得到 \(a_n = a_m + (n - m)d\) 这样比较普适的结论。

由等差数列求和公式推导的过程我们容易发现 \(m + n = p + q \Rightarrow a_m + a_n = a_p + a_q\)，这样得到了第二个性质，当 \(d \neq 0\) 时，我们也容易由右边推左边，这个结论还有一个特别的形式，即 \(m + n = 2p \Rightarrow a_m + a_n = 2a_p\)，这样我们就可以发现一种全新的等差数列求和方法，就是当项数 \(n\) 为奇数时，\(S_n = na_{\frac{n + 1}{2}}\)，这个结论可以帮助我们化简一些式子。

上述那个求和方式还可以逆用，比如 \(a_n = \dfrac{S_{2n - 1}}{2n - 1}\)，这一条性质可以解决一类问题，即给定等差数列 \(\{a_n\}\) 和 \(\{b_n\}\) 前 \(n\) 项和 \(S_n\) 和 \(T_n\) 的比值 \(\dfrac{S_n}{T_n}\)，求 \(\dfrac{a_n}{b_n}\)，这时候我们就可以利用 \(\dfrac{a_n}{b_n} = \dfrac{S_{2n - 1}}{T_{2n - 1}}\) 来进行计算。

等差数列的前 \(n\) 项和 \(S_n\) 也有一些很好的性质，下面我们来介绍一下。

首先根据 \(S_n = \dfrac{n(a_1 + a_n)}{2}\)，我们可以得到 \(\dfrac{S_n}{n} = \dfrac{a_1 + a_n}{2} = \dfrac{2a_1 + (n - 1)d}{2} = a_1 + \dfrac{n - 1}{2}d\)，也即 \(\left\{\dfrac{S_n}{n}\right\}\) 是一个公差为 \(\dfrac{d}{2}\) 的等差数列。

还有一个广为流传的结论是等差数列的片段和也是等差数列。形式化地，即 \(\{S_{nm} - S_{(n - 1)m}\}\) 也是个等差数列，现在我们不妨来研究一下这个等差数列的公差是什么。 \[ S_{(n + 1)m} - S_{nm} = \sum_{i={nm + 1}}^{(n + 1)m} a_i = \sum_{i={nm + 1}}^{(n + 1)m} a_{i - m} + md = m^2d + \sum_{i = (n - 1)m + 1}^{nm}a_i = m^2d + S_{nm} - S_{(n - 1)m} \] 所以 \(\{S_{nm} - S_{(n - 1)m}\}\) 是个公差为 \(m^2d\) 的等差数列。

等比数列

定义

如果数列 \(\{a_n\}\) 满足 \(a_n \neq 0\)，\(\dfrac{a_{n + 1}}{a_n}\) 为常数，那么则称 \(\{a_n\}\) 是一个等比数列，等比数列还可以写作 \(\text{G.P}\)。

等比数列的基本量

\(a_1\) 称为首项，\(\dfrac{a_{n + 1}}{a_n}\) 这个常数称为公比，一般用 \(q\) 表示。

和等差数列相似，我们容易得到等比数列的通项公式 \(a_n = a_1 \cdot q^{n - 1}\)。

等比数列的求和公式推导要用到一个错位相减的经典 trick，具体地，我们要求等比数列的前 \(n\) 项和 \(S_n = \sum_{i = 1}^n a_i\)，这个式子不是很好直接算，于是考虑消去求和式子中的一部分，考虑 \(qS_n = \sum_{i = 2}^{n + 1}a_i\)，两式相减，我们就可以得到 \((1 - q)S_n = a_1 - a_{n + 1}\)，所以 \(S_n = \dfrac{a_1 - a_{n + 1}}{1 - q} = \dfrac{(1 - q^n)a_1}{1 - q}(q \neq 1)\)。

等比数列的性质

等比数列的单调性不像等差数列那么直接，所以我们不研究它。

同等差数列相似，我们也可以对等比数列的通项公式进行变形，得到 \(a_n = a_m \cdot q^{n - m}\)。

还是同等差数列相似，\(m + n = p + q \Rightarrow a_ma_n = a_pa_q\)，特别地，\(m + n = 2p \Rightarrow a_ma_n = a_p^2\)。
