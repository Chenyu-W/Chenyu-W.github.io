发表于 2023-09-28 更新于 2023-09-30 分类于 笔记

孩子不懂事，写着玩的，轻喷。

从第二定义谈起

对于平面上的一个点 \(P(x, y)\)，如果它到一个顶点 \(F\) 的距离与到定直线 \(l\) 的距离的比值为 \(e\)，则称 \(P\) 的轨迹是圆锥曲线。其中 \(F\) 称为焦点，\(l\) 称为准线，\(e\) 称为离心率。

看到第二定义中的 \(e\)，有没有一点熟悉的感觉，没错，它就是椭圆和双曲线中的 \(e = \dfrac{c}{a}\)，那么抛物线呢？由定义可得，抛物线的 \(e = 1\)。下面给出一个表格。

方程 焦点 准线
\(\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1(a > b > 0)\) \((\pm c, 0)\) \(x = \pm\dfrac{a^2}{c}\)
\(\dfrac{x^2}{a^2} - \dfrac{y^2}{b^2} = 1(a>0,b>0)\) \((\pm c, 0)\) \(x = \pm \dfrac{a^2}{c}\)
\(y^2 = 2px(p > 0)\) \(\left(\dfrac{p}{2}, 0\right)\) \(x = -\dfrac{p}{2}\)

以上说的都是焦点在 \(x\) 轴的情况，焦点在 \(y\) 轴是一样的。

有了圆锥曲线第二定义，我们很容易就推导出来焦半径公式，比如在椭圆中就是 \(|PF_1| = a + ex_0\)，\(|PF_2| = e - ex_0\)，其余类似。

提到了第二定义，就不得不提到极坐标方程，以焦点为极点，\(x\) 轴为极线，记圆锥曲线焦点到准线的距离为 \(p\)，离心率为 \(e\)，那么我们可以得到圆锥曲线上一点\(P(\rho, \theta)\)到准线的距离为 \(\rho\cos\theta + p\)，所以我们可以得到 \(\dfrac{\rho}{\rho\cos\theta + p} = e\)，整理一下我们就可以得到圆锥曲线的统一极坐标方程为 \[ \rho = \dfrac{ep}{1 - e\cos\theta} \] 这个东西的作用大概是可以在处理一些与焦点弦相关的东西时设出方程，然后可以转到三角函数上进行计算或求极值。

参数方程

众所周知，单位圆上一点 \(P\) 可以被表示为 \(P(\cos\theta, \sin\theta)\) 的形式，我们类比一下，可以得到椭圆 \(\dfrac{x^2}{a^2} + \dfrac{y^2}{b^2} = 1\) 上一点 \(P\) 可以被表示为 \(P(a\cos\theta, b\sin\theta)\) 的形式，而 \(x = a\cos\theta\)，\(y = b\sin\theta\) 就称为椭圆的参数方程。类似的，我们可以推出双曲线和抛物线的参数方程。

下面我们来证明一个结论：

过椭圆上一点\(P(x_0, y_0)\) 的切线方程为 \(\dfrac{x_0x}{a^2} + \dfrac{y_0y}{b^2} = 1\)。

由椭圆的参数方程，我们不妨设 \(P(a\cos\alpha, b\sin\alpha)\)，这样直线的方程变为 \(\dfrac{x\cos\alpha}{a} + \dfrac{y\sin\alpha}{b} = 1\)，然后我们将这条直线和椭圆参数方程进行联立，就是将 \(x = a\cos\theta\)，\(y = b\sin\theta\)，这样就变为 \(\cos\theta\cos\alpha + \sin\alpha\sin\theta = 1\)，也就是 \(\cos(\theta - \alpha) = 1\)，因为 \(\theta, \alpha \in [0, 2\pi)\)，所以我们可以得到这个方程有唯一解 \(\theta = \alpha\)，也就是它是椭圆的切线。

这个结论拓展一下，我们可以得到过椭圆外 \(P(x_0, y_0)\) 的切点弦方程为 \(\dfrac{x_0x}{a^2} + \dfrac{y_0y}{b^2} = 1\)。这样你就可以秒掉和切线相关的题目了！

圆锥曲线的标准方程还可以和直线的参数方程 \(x = x_0 + t\cos\theta\)，\(y = y_0 + t\sin\theta\) 进行联立，在求解一类线段长问题可能会有用处。

圆锥曲线方程的变形

看知乎一篇文章学的，忘记是哪篇了。

在圆锥曲线问题中，有一类问题长这样：

过圆锥曲线一点 \(P(x_0, y_0)\) 做两条斜率分别为 \(k_1, k_2\) 的直线，满足 \(k_1 + k_2\) 或 \(k_1k_2\) 为定值，两直线分别另交圆锥曲线于 \(A,B\)，证明直线 \(AB\) 过定点（或告诉 \(AB\) 过顶点，求 \(k_1, k_2\) 的关系）。

下面我们以椭圆为例，其他是类似的。

首先有一个常见结论，给定椭圆上关于原点对称的两点 \(A, B\)，那么对于椭圆上一点 \(P\)，都有\(k_{PA}k_{PB} = -\dfrac{b^2}{a^2}\)，所以我们可以把椭圆的方程写成 \(\dfrac{(y - y_0)(y + y_0)}{(x - x_0)(x + x_0)} = -\dfrac{b^2}{a^2}\)，其中 \((x_0, y_0)\) 是椭圆上一点。

然后我们开始证明这题。

设 \(PA: y - y_0 = k_1(x - x_0)\)，\(PB:y - y_0 = k_2(x - x_0)\)，我们希望快速得到两条直线上所有点的方程，所以我们可以把两条直线的方程乘起来得到

\[ k_1k_2(x - x_0)^2 - (k_1 + k_2)(x - x_0)(y - y_0) + (y - y_0)^2 = 0 (1) \]

然后我们对一开始写的方程变形得到 \(y - y_0 = -\dfrac{b^2(x - x_0)(x + x_0)}{a^2(y + y_0)}\)，在将这个式子代到上面的方程里面得到 \[ k_1k_2(x - x_0)^2 + (k_1 + k_2)(x - x_0)\dfrac{b^2(x - x_0)(x + x_0)}{a^2(y + y_0)} + \left[\dfrac{b^2(x - x_0)(x + x_0)}{a^2(y + y_0)}\right]^2 = 0 \] 整理一下可以得到 \[ \dfrac{a^2}{b^2}k_1k_2(y + y_0)^2 + (k_1 + k_2)(x + x_0)(y + y_0) + \dfrac{b^2}{a^2}(x + x_0)^2(2) \] 然后我们惊奇地发现如果把 \((1)\) 和 \((2)\) 加起来，不仅 \(xy\) 项的系数消掉了，而且会出现 \(\dfrac{a^2}{b^2}y^2 + x^2\) 和 \(\dfrac{b^2}{a^2}x^2 + y^2\) 这两项，根据椭圆的方程，我们可以得到这两个都是常数，所以我们加起来实际上获得了一个一次的东西！

所以我们进行 \((1) + (2)\)，化简即可得到。 \[ AB:k_1k_2\left(\dfrac{a^2}{b^2}y_0y - x_0x + a^2\right) + (k_1 + k_2)(x_0y + y_0x) + \left(\dfrac{b^2}{a^2}x_0x - y_0y + b^2\right) = 0 \] 到这一步就不难看出 \(AB\) 过定点了。

注意当 \(k_1k_2 = \dfrac{a^2}{b^2}\) 或 \(k_1 + k_2 = 0\) 时结论变为 \(AB\) 斜率一定。

双曲线是类似的，只需记住方程可写作 \(\dfrac{(y - y_0)(y + y_0)}{(x - x_0)(x + x_0)} = \dfrac{b^2}{a^2}\) 即可。

抛物线是 \(x - x_0 = \dfrac{1}{2p}(y + y_0)(y - y_0)\)。

这种方法可以解决一类问题，理解其思路即可。
