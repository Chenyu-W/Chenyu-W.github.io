卷积（FFT&NTT&MTT）

基本思路

先将多项式转化为点值表达，点值对应相乘后再插值回系数表达。

单位根

首先引入 \(n\) 次单位根，即在复平面上满足 \(x^n = 1\) 的 \(n\) 个根，记第 \(k\) 个根为 \(w_n^k = \cos(\frac{2k\pi}{n}) + i\sin(\frac{2k\pi}{n})\)。

\(n\) 次单位根有一些重要的性质。

  1. \(w_n^1, w_n^2, \ldots, w_n^n\) 两两不同（每一个 \(w\) 可以看做前一个旋转了 \(\dfrac{2\pi}{n}\)）。
  2. \(w_{2n}^{2k} = w_{n}^k\)。
  3. \(w_{n}^{k + n / 2} = -w_{n}^k\)（可看作旋转了 \(\pi\)）。
  4. \((w_{n}^{k})^2 = w_{n/2}^k\)

这些性质是单位根能被利用于处理多项式卷积的关键。

DFT

这一步把多项式转化为点值表达，朴素的多点求值时间复杂度为 \(O(n^2)\)，我们尝试在单位根处求值以降低时间复杂度。

首先考虑次数为 \(n = 2^m - 1\) 的多项式 (不够在最高次补 \(0\)）\(f(x) = \sum a_ix^i\)，我们要计算的即为 \(f(w_n^1),\ldots,f(w_n^n)\)。我们将 \(f(x)\) 作以下变形。 \[ f(x) = \sum_{i = 0}^n a_ix^i = (a_0 + \cdots + a_{n - 1}x^{n - 1}) + (a_1x + \cdots + a_nx^n)\\ f_0(x) = (a_0 + a_2x + \cdots + a_{n - 1}x^{(n - 1)/2})\\ f_1(x) = (a_1 + a_3x + \cdots + a_{n}x^{(n - 2) / 2})\\ f(x) = f_0(x^2)+xf_1(x^2) \] 将 \(n\) 个单位根带入，我们可以得到 \[ f(w_n^k) = f_0(w_{n/2}^k) + w_n^kf_1(w_{n/2}^k)\\ f(w_n^{k + n / 2}) = f_0(w_{n/2}^k) - w_n^kf_1(w_{n/2}^k) \] 于是我们只要递归下去分别求 \(f_0(x)\) 与 \(f_1(x)\) 的点值就可以得到一个 \(O(n\log n)\) 的算法了。

但递归下去常数过大，我们考虑非递归的时限，我们把递归树建出来，可以得到类似如下的变化 \[ (0, 1, 2, 3) \to (0, 2), (1, 3) \to (0),(2),(1),(3) \] 我们发现每个数所处的最终位置是他二进制位的逆序，所以我们可以预处理每个位置最终在递归树的位置，然后自底向上合并即可。

IDFT

得到点值表达后，我们需要将其插值为系数表达，IDFT 完成的就是这一步。

我们用矩阵乘法来刻画 DFT 的过程，发现实质上我们在做一个这样的矩阵乘法。 \[ \begin{bmatrix} 1 & w_n^1 & \cdots & (w_n^1)^n\\ 1 & w_n^2 & \cdots & w_n^2 \\ \vdots & \vdots & \ddots & \vdots \\ 1 & w_n^n & \cdots & (w_n^n)^n \end{bmatrix} \begin{bmatrix} a_0\\ a_1\\ \vdots\\ a_n \end{bmatrix} = \begin{bmatrix} y_0\\ y_1\\ \vdots\\ y_n \end{bmatrix} \] 于是我们想要还原，只需要得到最左侧单位根矩阵的逆矩阵即可。

我们发现左侧是一个范德蒙德矩阵，其逆矩阵为每一项取倒数再乘 \(\dfrac{1}{n}\)，于是我们只要将单位根 \(w_n^k\) 换成 \((w_n^k)^{-1}\)，再做一遍 DFT，即可得到原多项式的系数表达。

NTT

FFT 的精度和常数显然不是很好，对于取模 \(p = q \times 2^k + 1\)（\(k\) 足够大）的情况下，我们可以使用 NTT 解决。

我们记 \(g\) 为原根，\(w_n^k = g^{(p-1)k/n} \bmod p\)，用 \(w_n^k\) 来代替单位根即可做到同样的效果。

MTT

NTT 只能处理模数固定的多项式乘法，如果模数为任意值就要使用 MTT 了。

设模数为 \(p\)，我们取一个在 \(\sqrt p\) 左右的阈值 \(q\)，将多项式的每一项拆成 \(aq + b\) 的形式，最后把几个结果拼起来即可。

多项式牛顿迭代

基本原理

众所周知，\(f(x) = 0\) 的近似根可以通过牛顿迭代求得，那么 \(g(f(x)) \equiv 0 \pmod{x^n}\) 的根如何求呢？

我们考虑倍增法求解，首先求出 \(g(f(x)) \equiv 0\pmod{x^1}\) 的根 \(f_0(x)\)，然后我们考虑倍增求解之后的根。

假设我们知道 \(g(f(x)) \equiv 0\pmod{x^n}\) 的根 \(f_0(x)\)，我们尝试取求出 \(g(f(x)) \equiv 0 \pmod{x^{2n}}\) 的根 \(f(x)\)。

对 \(g(f(x))\) 在 \(f_0(x)\) 处取泰勒展开得到 \[ g(f(x)) \equiv \sum_{k = 0}^{\infty} \dfrac{g^{(k)}(f_0(x))}{k!}(f(x) - f_0(x))^k \pmod{x^{2n}} \] 由于上述式子是在模 \(x^{2n}\) 意义下进行的，所以 \(k \ge 2\) 的项可以全部忽略，我们得到 \[ g(f(x)) \equiv g(f_0(x)) + g'(f_0(x))(f(x)-f_0(x)) \equiv 0 \pmod{x^{2n}} \]

最终得到 \[ f(x) \equiv f_0(x) - \dfrac{g(f_0(x))}{g'(f_0(x))} \pmod{x^{2n}} \]

多项式求逆

给定多项式 \(h(x)\)，求多项式 \(f(x)\) 使的 \(f(x)h(x) \equiv 1 \pmod{x^n}\)。

记 \(g(f(x)) = h(x) - \dfrac{1}{f(x)}\)，那么即求 \(g(f(x)) \equiv 0 \pmod{x^n}\) 的根。

然后套用上述公式，我们得到 \[ f(x) \equiv f_0(x)(2 - f_0(x)h(x)) \pmod{x^{2n}} \]

直接倍增即可。

时间复杂度 \(T(n) = T\left(\dfrac{n}{2}\right) + O(n\log n) = O(n\log n)\)。

多项式开方

给定多项式 \(h(x)\)，求多项式 \(f(x)\) 使得 \(f^2(x)\equiv h(x)\pmod{x^n}\)。

令 \(g(f(x)) = h(x) - f^2(x)\)，同上得到 \[ f(x)\equiv \dfrac{f_0(x) + h(x)f_0^{-1}(x)}{2} \pmod{x^{2n}} \] 边倍增边求逆即可，初解可以用二次剩余求得，时间复杂度 \(O(n\log n)\)。

多项式除法

给定次数为 \(n, m (n\ge m)\) 的多项式 \(f(x),g(x)\)，求多项式 \(q(x), r(x)\)，满足 \(f(x) = q(x)g(x) + r(x)\)，且 \(\deg q = n - m\)，\(\deg r < m\)。

我们发现如果把 \(r(x)\) 消去可以直接用多项式求逆解决，于是有一个很神奇的方法出现了。

将等式两边的 \(x\) 用 \(\dfrac{1}{x}\) 代替，再乘上 \(x^n\)，得到 \[ f\left(\dfrac{1}{x}\right)x^n = g\left(\dfrac{1}{x}\right)x^mq\left(\dfrac{1}{x}\right)x^{n-m}+r\left(\dfrac{1}{x}\right)x^{m - 1}x^{n - m + 1} \] 然后我们将发现上述操作实际上是将系数反转了，我们把这个多项式记为 \(f_R(x)\)，得到 \[ f_R(x) = g_R(x)q_R(x) + r_R(x)x^{n - m + 1} \] 因为 \(\deg q < n - m + 1\)，所以我们可以直接把 \(r(x)\) 消去，得到 \[ q_R(x) \equiv f_R(x)g_R^{-1}(x)\pmod{x^{n - m + 1}} \] 知道了 \(q(x)\)，即可解出 \(r(x)\)。

多项式指对函数

首先考虑求 \(f(x) \equiv \ln h(x) \pmod{x^n}\)，我们考虑等式两边同时求导再积分，可以得到 \[ f(x) \equiv \int \dfrac{h'(x)}{h(x)} \pmod{x^n} \]

于是多项式求逆即可 \(O(n\log n)\) 解决。

注意 \(\ln\) 只在 \([x^0]f(x) = 1\) 时有意义。

然后考虑求 \(f(x) \equiv e^{h(x)} \pmod{x^n}\)，令 \(g(f(x)) = h(x) - \ln f(x)\)，我们用牛顿迭代可以得到 \[ f(x) \equiv f_0(x) - \dfrac{g(f_0(x))}{g'(f_0(x))} = f_0(x)(1 - \ln f_0(x) + h(x)) \pmod{x^{2k}} \] 于是倍增可以 \(O(n\log n)\) 求解。

注意 \(\exp\) 只在 \([x^0]f(x) = 0\) 有意义。
